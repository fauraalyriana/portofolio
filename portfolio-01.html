ABC LEARNING CENTER

Project information

  • Category: Web design
  • Project date: 24 November, 2022
  • Source Code: ABC Learning Center
  • Project Demo: View demo

Project ini dikerjakan saat saya mengikuti bootcamp. Project ini bersifat individu, dimana kita diminta untuk merancang dan membuat situs website untuk ABC LEARNING Center dengan menggunakan HTML, CSS dan Javascript. ABC Learning Center adalah lembaga Pelatihan IT yang menyelenggarakan kursus seperti Pemrograman di Java, Pemrograman di .Net, Microsoft Office, HTML & dll. Situs web yang dapat digunakan untuk memberikan informasi tentang mereka kursus bagi siswa dan untuk menangkap prospek yang dihasilkan oleh pemasaran digital.

  • Berhasil menyelesaikan 10 halaman
  • Berhasil menerapkan technology yang sesuai dengan ketentuan project

    • Dalam mengerjakan project ini, saya menggunakan HTML sebagai dasarnya, membangun strukturnya halaman dengan semantik. Kemudian dengan sentuhan CSS, saya mendekorasi situs untuk menciptakan pengalaman pengguna yang menyenangkan. Saya juga gunakan Javascript untuk menambahkan elemen interaktif ke situs.
